# MiniProgram
基于云开发的微信小程序

简介
----

- 基于小程序云开发，实现结伴小程序徐
- 采用组件化开发，实现登录组件，验证组件，底部弹出层，博客卡片
- 云数据库，使用 tcb-router 实现路由管理
- 使用模板推送实现分享功能
- 点开博客卡片可以加入结伴列表
- 个人中心可以查看历史记录

依赖
-----

- node.js
- tcb-router

如何运行
---------

- 打开微信开发者工具
- 运行此项目
